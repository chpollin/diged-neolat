1. Navigating the Edition

The Lucina Digital Edition provides multiple ways to navigate through the text:

Book and Poem Selection

  • Use the Book dropdown in the navigation bar to select a specific book (1-4)
  • Use the Poem dropdown to jump directly to any poem
  • The dropdowns update dynamically based on your selection

Page Navigation

  • Use the arrow buttons ‚Üê ‚Üí to navigate between manuscript pages
  • The current page number is displayed in the image viewer header
  • Scroll through the text panel to read continuously
üí° Tip: The text and manuscript image stay synchronized as you navigate. Selecting a poem automatically displays the corresponding manuscript page.

2. Different Views

Switch between three different viewing modes using the toggle buttons in the center of the navigation:

Latin View

The primary Latin text from the manuscript with line numbers and critical apparatus.

Commentary View

Scholarly commentary and notes on specific passages, references, and interpretations.

Translation View

Modern language translations of the Latin text for easier comprehension.

3. Search Features

Basic Search

  • Type in the search box to find text anywhere in the edition
  • Search results are highlighted in yellow
  • Use the navigation arrows to jump between results
  • The result counter shows current position and total matches

Advanced Search Tips

  • Search is case-insensitive by default
  • Searches across all text content including apparatus
  • Clear the search box to remove highlighting

5. Annotations & Commentary

Types of Annotations

  • Person References: Highlighted names link to prosopographical information
  • Place References: Geographic locations with modern identifications
  • Critical Apparatus: Textual variants and editorial notes
  • Line-specific Commentary: Scholarly notes on individual lines

Interacting with Annotations

  • Hover over highlighted text to see quick information
  • Click on references to see full details
  • Commentary appears in the dedicated Commentary view

Plain Text

8. Edit Mode (Advanced)

‚ö†Ô∏è Note: Edit Mode is only available when running the edition locally. It is disabled in the online version to protect the integrity of the published edition.

What is Edit Mode?

Edit Mode allows scholars and editors to make corrections and annotations directly in the browser, saving changes as new TEI XML files.

How to Enable Edit Mode

  1. Download the edition files to your local computer
  2. Open locally using one of these methods:
    • Double-click index.html to open in your browser
    • Or start a local web server:
    cd docs/edition-5/web
python -m http.server 8000
# Visit: http://localhost:8000
  3. Click the Edit Mode button to enable edit mode
  4. Make your edits directly in the text
  5. Save changes using the Save button

Edit Mode Features

  • Direct text editing with change tracking
  • Attribute modification for TEI elements
  • Diff view to review changes before saving
  • XML validation
  • Export edited version as new TEI file
  • Revert changes if needed

Why is Edit Mode Restricted?

Edit Mode is intentionally limited to local environments to:

  • Prevent accidental modifications to the published edition
  • Ensure editorial changes go through proper review
  • Maintain the scholarly integrity of the digital edition
  • Allow controlled collaborative editing workflows
